# TaskForge

**TaskForge** is a distributed task scheduling system built in Java. It provides scalable and fault-tolerant task execution across worker nodes with leader election and coordination mechanisms.

## 📦 Package Structure

- `core` – Core interfaces and logic for task scheduling
- `worker` – Logic for worker node registration and task execution
- `coordinator` – Leader election and coordination strategies
- `api` – RESTful APIs for task submission and monitoring
- `persistence` – Data access and durable storage handling
- `common` – Shared utilities and constants

## 🚀 Features

- Distributed task scheduling with coordination
- Leader election for failover and reliability
- RESTful API for task submission and status
- Modular, extensible architecture
